This project is a browser-based 3D audio visualization tool that lets users upload an audio file and explore the same sound in multiple visual forms. The main visualization will be a grid or surface made of points with X, Y, and Z coordinates that animate in real time while the audio plays. Users will be able to choose what each axis represents, such as frequency range, time or playback position, stereo position, waveform position, amplitude, or frequency-band intensity. The selected audio properties will determine where points are placed and how high they rise, while mathematical effects such as sine and cosine waves can be added to create smoother or more expressive movement. The interface should include audio playback controls, axis-selection menus, visualization settings, and the ability to rotate, zoom, and inspect the 3D scene. The goal is to create an interactive audio-analysis playground where users can experiment with different mappings and understand how properties such as volume, pitch, frequency, time, and left-right stereo balance affect the appearance of sound. The project will primarily use React for the interface, the Web Audio API for real-time audio analysis, and Three.js—likely through React Three Fiber—for rendering the animated 3D visualization.
